📋 项目概述

免疫浸润分析工具是一个基于 R 语言开发的多算法免疫细胞浸润分析平台。整合了 8 种主流的免疫浸润分析方法，提供统一的数据输入接口和标准化的分析流程。

核心功能

  • ✅ 支持 8 种主流免疫浸润分析算法
  • ✅ 自动进行统计差异分析（Wilcoxon 检验）
  • ✅ 支持基因表达与免疫细胞相关性分析
  • ✅ 自动生成堆叠柱状图和箱线图
  • ✅ 支持缓存机制，避免重复计算
  • ✅ Python 主控脚本支持并行执行

🔬 分析方法

方法输出类型细胞类型数特殊要求

CIBERSORT

原理：基于支持向量回归（SVR）的反卷积算法，估算 22 种免疫细胞类型的相对比例。

适用场景：肿瘤微环境分析、免疫细胞比例量化

特殊参数：-p/--perm 排列次数（默认 1000）

xCell

原理：基于基因集富集的细胞类型评分方法，涵盖 64 种细胞类型。

适用场景：广泛的细胞类型分析、基质细胞评估

特殊参数：--arrays 是否为芯片数据

ESTIMATE

原理：基于基因签名计算肿瘤微环境评分。

输出：ImmuneScore、StromalScore、ESTIMATEScore

适用场景：肿瘤纯度评估、微环境整体评价

ssGSEA

原理：单样本基因集富集分析。

适用场景：自定义免疫细胞签名分析

特殊参数：--signature 基因签名文件（必需）

TIMER

原理：基于约束反卷积的免疫浸润估算。

输出：6 种主要免疫细胞浸润水平

特殊参数：--tissue 肿瘤类型（必需，如 LUAD、BRCA）

EPIC

原理：使用约束最小二乘法估算细胞比例。

输出：8 种细胞类型的绝对比例

适用场景：肿瘤与正常组织比较

MCPcounter

原理：基于标记基因的微环境细胞种群计数。

输出：10 种细胞类型的丰度评分

适用场景：多种细胞类型的相对丰度比较

IPS

原理：免疫表型评分，评估免疫治疗响应潜力。

输出：MHC、EC、SC、CP 四个维度的评分

适用场景：免疫治疗响应预测

🚀 快速开始

方式一：使用 R 脚本

# 进入脚本目录
cd immune_infiltration/main/scripts

# 运行 CIBERSORT 分析
Rscript cibersort.R \
  --expr ../../test/input/expr_tpm.csv \
  --group ../../test/input/group.csv \
  --output ../../test/output/CIBERSORT \
  --perm 1000

# 运行 xCell 分析
Rscript xcell.R \
  --expr ../../test/input/expr_tpm.csv \
  --group ../../test/input/group.csv \
  --output ../../test/output/xCell

方式二：使用 Python 主控脚本

# 修改配置文件
vim config/config.ini

# 运行所有方法
python main/scripts/run_immune_infiltration.py --all

# 运行指定方法
python main/scripts/run_immune_infiltration.py \
  --methods cibersort xcell estimate \
  --workers 3

# 并行运行（使用 4 个进程）
python main/scripts/run_immune_infiltration.py --all --workers 4

📥 输入格式

1. 表达矩阵 (expr_tpm.csv)

CSV 格式，第一列为基因符号（SYMBOL），其余列为样本表达值。

SYMBOL,Sample1,Sample2,Sample3,…
TP53,12.5,15.3,11.8,…
EGFR,8.2,9.1,7.5,…
MYC,25.6,28.4,22.1,…
...

2. 分组信息 (group.csv)

CSV 格式，包含样本 ID 和分组信息。

ID,group
Sample1,Tumor
Sample2,Tumor
Sample3,Normal
Sample4,Normal
...

3. 基因列表 (可选)

用于相关性分析，CSV 格式，包含 gene 列。

gene
TP53
EGFR
MYC
...

4. 签名文件 (ssGSEA 专用)

GMT 格式或两列 CSV 格式。

CellType,Gene
T_cells,CD3D
T_cells,CD3E
B_cells,CD19
B_cells,MS4A1
...

❓ 常见问题

Q1: 找不到 group_color 变量？

确保加载了正确的函数库：source("../lib/immune_functions.R")

Q2: CIBERSORT 运行很慢？

CIBERSORT 默认进行 1000 次排列，可以使用 -p 100 减少排列次数进行快速测试。正式分析建议使用 1000。

Q3: TIMER 需要 tissue 参数？

TIMER 方法需要指定肿瘤类型缩写，如：
  • LUAD - 肺腺癌
  • BRCA - 乳腺癌
  • COAD - 结肠癌
  • PRAD - 前列腺癌

Q4: 如何选择分析方法？

  • 细胞比例：CIBERSORT、EPIC
  • 富集评分：xCell、ssGSEA、MCPcounter
  • 整体评分：ESTIMATE
  • 免疫治疗：IPS
  • 推荐：多方法联合使用，取交集验证

Q5: 缓存机制如何工作？

结果会缓存到 cache/ 目录。如需重新计算，使用 --force 参数或删除缓存文件。
